# Instructions for setting up Data Knot

## How to start:
1) install the FluCoMa package via the Package Manager in Max or via https://www.flucoma.org/download/
2) install Data Knot package via the Package Manager in Max or via http://github.com/rconstanzo/data-knot

You can then open the Data Knot Overview patch via the Package Manager info window, or from the extras menu bar.
