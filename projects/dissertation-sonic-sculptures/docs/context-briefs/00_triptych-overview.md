# Organic Memory (Triptych) — Master Context Brief

## Project Identity
**Full title:** Organic Memory (Triptych)
**Dissertation framework:** Sonic Speculocultural Technopoiesis (ST)
**Researcher:** Darren Woodland Jr. (DJ), Ph.D. candidate, Drexel University Westphal College
**Status:** IRB approved, committee approved, active creation phase (Feb 2026)

## What This Is
Three instruments — each discrete and self-contained — that assemble into a single unified instrument, played by one performer (the researcher/builder), enclosed by a spatial audio array. The three works are simultaneously:
- **Three instruments** — each with its own housing, material basin, sensing ecosystem, circuit logic, and performative affordances
- **One instrument** — modular, assembled into a shared performance console for the composition
- **One composition** — *Organic Memory (Triptych)*, performed in sequence through the three modules
- **One dissertation** — structured organically (non-formulaic) as the score for the instrument/composition
- **Methodological demonstrations** of the ST framework as research instruments

**David Tudor model:** The instrument is built to be played only by its maker. The performer's embodied and cultural knowledge is inseparable from the instrument itself — this is not a limitation but the epistemological claim. No one else can perform this work identically, and that irreducibility is the point.

**Spatial layout:** The physical arrangement of the three modules relative to each other and the performer's body is still open — to be resolved through sketching and physical mockup. Draw the performer's body first; build the instrument around it. The speaker array encloses the performer, not the audience.

## The ST Cycle (applied to each work)
1. **Unsettling** — identifying cultural friction with existing tools
2. **Remixing** — embodied modification of tools/parameters
3. **Embodying** — testing through practice, cultural authenticity over technical correctness
4. **Transmitting** — documenting so others can adapt with cultural reasoning intact

## Four ST Domains (analytical filters applied throughout)
- **Fugitive Speculation (FS)** — refuses dominant narratives, imagines otherwise
- **Material Memory (MM)** — objects/media as cultural archives
- **Sensory Worlding (SW)** — multisensory, non-ocularcentric knowing
- **Intimate Witnessing (IW)** — vulnerable sharing, cultural validation

## The Shared Motif
A rhythmic/melodic cell derived from the final phrase of *When Johnny Comes Marching Home* (also *The Ants Go Marching*):

**[triplet pickup] + [short-short-short-short-LONG]**

This cell threads through all three movements, transformed by each work's material and temporal logic:
- Buffalo Soldier → geological (stretched, subsonic, felt not heard)
- Did Sun Ra Fear Cryosleep? → aquatic (dissolved, fragmented by chemistry)
- Don't Play with My Hair → intimate (human scale, most recognizable, clipper-adjacent)

**Why this motif:** Civil War era, contemporaneous with Buffalo Soldiers; ironic triumphalism against Black soldiers' actual reception; 6/8 compound meter connects European march tradition to African diasporic rhythmic DNA; short enough to be a cell, strong enough to survive transformation.

## The Sleigh Bells (Eastman Reference)
Inspired by Julius Eastman's *Femenine* — a low buzzing drone tuned to the motif's long note, present at threshold of audibility throughout all three movements. Never fully absent. The condition of the triptych's world, not a foreground element.

In each movement: subsonic rumble (Buffalo Soldier) → vessel resonance (Did Sun Ra) → clipper hum (Don't Play with My Hair).

## AI Layer: Somax2
Each movement uses Somax2 (Max/MSP) for corpus-based co-improvisation. Somax2 is trained on culturally specific corpora per movement and responds to material sensor input by elaborating from those corpora. The shared motif is the seed pattern Somax2 recognizes across all three.
